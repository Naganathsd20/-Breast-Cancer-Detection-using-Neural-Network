# 🩺 Breast Cancer Detection using Neural Network

> An AI-powered classification model that predicts whether a breast tumor is **Benign** or **Malignant** using a Deep Learning Neural Network.

---

## 📌 Project Overview

Early detection of breast cancer plays an important role in improving treatment outcomes. This project uses a **Neural Network** built with **TensorFlow** and **Keras** to classify breast tumors based on medical diagnostic features.

The model is trained on the Breast Cancer Wisconsin Dataset and can accurately predict whether a tumor is **Benign (Non-Cancerous)** or **Malignant (Cancerous)**.
